# Module: Mobile

## 1. 模块定位

Capacitor 移动端壳体，承载 Web 构建产物并提供原生能力。

本模块职责：

- Capacitor 配置与原生工程管理
- iOS / Android 构建与发布
- 移动端权限控制（麦克风、通知）
- 系统级推送提醒集成

不属于本模块：

- Web UI（Web 模块构建产物）
- HTTP 服务（API 模块）
- 语音解析（Voice + Parser 模块）

## 2. 产品目标与范围

### 目标

- 基于 Web 已验证的能力，通过 Capacitor 在移动端提供更自然、更高频的语音日程体验

### 范围内

- Capacitor 壳体与移动端发布能力
- 受移动端权限控制的语音交互
- 推送提醒与系统级集成
- 更适合碎片场景的移动端交互

### 范围外

- 独立原生 UI（复用 Web 构建产物）
- 多端实时同步（首期复用 Web 数据层）

## 3. 接口契约

- **暴露**：App 安装包（ipa/apk）
- **依赖**：Web（构建产物目录）、API（部署后的 HTTP 服务）
